Basic Things Every Beginner Should Know

Steps to Create a Gmail Account

  1. Go to the Gmail website.
  2. Click on "Create account".
  3. Fill in your personal information.
  4. Choose a username and password.
  5. Verify your phone number.
  6. Set up recovery options.
  7. Accept the terms and conditions.
  8. You're done! You can now use your Gmail account.

Steps to Create a GitHub Account

  1. Go to the GitHub website.
  2. Click on "Sign up".
  3. Enter your email address.
  4. Create a password.
  5. Choose a username.
  6. Verify your email address.
  7. Complete the setup steps.
  8. You're done! You can now use your GitHub account.

Steps to Clone/Pull a Repo in GitHub

  1. Open GitHub and navigate to the repository you want to clone.
  2. Click on the "Code" button.
  3. Copy the URL provided.
  4. Open your terminal and navigate to the directory where you want to clone the repo.
  5. Type git clone [URL] and press Enter.
  6. To pull changes, navigate to the repository folder in your terminal.
  7. Type git pull and press Enter.
  8. You're done! You've cloned and pulled a repository from GitHub.
