Functions in C

Functions in C can be categorized into four types based on their return type and arguments.

Types of Functions

  • Return Type with No Argument

    This type of function returns a value but does not take any arguments. Example: Addition

    int add() {
    int a = 5, b = 10;
    return a + b;
}

  • Return Type with Argument

    This type of function returns a value and takes arguments. Example: Division

    float divide(int a, int b) {
    return (float)a / b;
}

  • No Return Type with Argument

    This type of function does not return a value but takes arguments. Example: Subtraction

    void subtract(int a, int b) {
    printf("Difference: %d", a - b);
}

  • No Return Type with No Argument

    This type of function neither returns a value nor takes arguments. Example: Multiplication

    void multiply() {
    int a = 5, b = 10;
    printf("Product: %d", a * b);
}
