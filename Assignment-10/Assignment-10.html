NOTES AND LAB REPORTS OF FILE HANDLING IN C

FILE HANDLING IN C

COMMON TERMINOLOGIES IN FILE HANDLING

  • FILE: A location or object where we store data in various forms, such as video, audio, text, image, etc.
  • TEXT FILE: Files containing readable characters and formatted text (e.g., ASCII or UTF-8). Operations include fopen, fgets, fprintf, fclose.
  • BINARY FILE: Files with data in non-readable format, stored as raw bytes. Operations include fopen, fread, fwrite, fclose.

fgetc Function

Syntax:
    int fgetc(FILE *stream);

Description:
    - Reads the next character from the specified file stream.
    - Returns the character read as an unsigned char cast to an int or EOF if the end of the file is reached or an error occurs.

Usage Example:
    FILE *file = fopen("example.txt", "r");
    if (file) {
        int ch;
        while ((ch = fgetc(file)) != EOF) {
            putchar(ch); // Output each character to standard output
        }
        fclose(file);
    } else {
        perror("Error opening file");
    }

fputc Function

Syntax:
    int fputc(int char, FILE *stream);

Description:
    - Writes the character (an unsigned char cast to an int) to the specified file stream.
    - Returns the character written as an unsigned char cast to an int or EOF if an error occurs.

Usage Example:
    FILE *file = fopen("example.txt", "w");
    if (file) {
        fputc('H', file); // Write character 'H' to the file
        fputc('e', file); // Write character 'e' to the file
        fputc('l', file); // Write character 'l' to the file
        fputc('l', file); // Write character 'l' to the file
        fputc('o', file); // Write character 'o' to the file
        fclose(file);
    } else {
        perror("Error opening file");
    }
