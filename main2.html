Keywords or Commands Used in SQL to Imply Change in DBMS

1. CREATE DATABASE

Used to create a new database.

Syntax:

CREATE DATABASE database_name;

Example:

CREATE DATABASE STUDENT;

Output:

2. CREATE TABLE

Used to create a new table in the database.

Syntax:

CREATE TABLE table_name(column_name datatype, column_name datatype, column_name datatype ...);

Example:

CREATE TABLE HOMEWORK (ID INT(3) PRIMARY KEY AUTO_INCREMENT, NAME VARCHAR(255), ADDRESS VARCHAR(255), STAT VARCHAR(255), GENDER VARCHAR(30));

Output:

3. ALTER

Used to alter or modify new records into a table.

Syntax:

ALTER TABLE table_name
ADD COLUMN column_name INT(x);

Example:

ALTER TABLE HOMEWORK
ADD COLUMN AGE INT(3);

Output:

4. ADD

Used to add new datas like field in table.

Syntax:

ALTER TABLE table_name
ADD COLUMN column_name INT(x);

Example:

ALTER TABLE HOMEWORK
ADD COLUMN AGE INT(3);

Output:

5. SELECT

Used to retrieve data from a database.

Syntax:

SELECT column_name, column_name, column_name, ... FROM table_name;

Example:

SELECT * FROM HOMEWORK;

Output:

6. UPDATE

Used to modify existing records in a table.

Syntax:

UPDATE table_name SET column_name = value, column_name = value, ... WHERE condition;

Example:

UPDATE HOMEWORK SET PHONE = 1234567890 WHERE ROLLNO = 1;

Output:

7. DELETE

Used to delete existing records from a table.

Syntax:

DELETE TABLE table_name;

Example:

DELETE TABLE HOMEWORK;

Output:

8. INSERT INTO

Used to insert new records into a table.

Syntax:

INSERT INTO table_name(column_name, column_name, column_name, ...) VALUES(value1, value2, value3, ...);

Example:

INSERT INTO HOMEWORK(ROLLNO, NAME, ADDRESS, PHONE) VALUES(1, 'Rahul', 'Delhi', 1234567890);

Output:
